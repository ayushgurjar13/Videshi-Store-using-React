# Videshi Store

An e-commerce web application built with React.

## About

This is a simple online shopping platform where users can browse products, filter them by category, search by name, and manage their shopping cart.

## Features

- Product listing with images, prices, and stock status
- Search products by name
- Filter by category
- Sort by price (low to high / high to low)
- Add/remove items from cart
- Update product quantity in cart
- Cart page for viewing all cart items
- Cart data saved in browser

## Technologies Used

- React.js
- React Router
- CSS
- DummyJSON API for product data

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the application:
```bash
npm start
```

3. Open browser and go to `http://localhost:3000`
